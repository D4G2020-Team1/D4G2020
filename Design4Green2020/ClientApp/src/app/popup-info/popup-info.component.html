Infos sur l'application

Cette application intègre les fonctionnalités suivantes :

  • Sélection/recherche d'une région pour filtrage
  • Sélection/recherche d'un département pour filtrage
  • Sélection/recherche d'une intercommunalité pour visualiser les zones en gradient sur la carte
  • Sélection/recherche d'une commune pour visualiser les zones en gradient sur la carte
  • Survol d'un quartier affiché sur la carte pour obtenir un tooltip d'information sur le score du quartier
  • Clic sur un quartier affiché sur la carte pour obtenir son score
  • Export PDF du résultat sur le quartier sélectionné
  • Enregistrement en local de la dernière recherche réalisé
  • Export des statistiques de recherche de score
